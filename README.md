# Users CRUD PHP

Um sistema simples de CRUD (Create, Read, Update, Delete) de usuários desenvolvido em PHP, utilizando HTML, CSS, SCSS e JavaScript na interface.

## 📚 Descrição

Este projeto foi criado com o objetivo de praticar e demonstrar um CRUD completo de usuários. Ele permite cadastrar, listar, editar e remover usuários, utilizando PHP para backend e um frontend responsivo com CSS/SCSS, HTML e algumas funcionalidades em JavaScript.

## 🛠️ Tecnologias Utilizadas

- **PHP**: Backend e lógica do CRUD
- **MySQL**: Banco de dados para armazenamento dos usuários
- **CSS / SCSS**: Estilização da interface
- **HTML**: Estrutura das páginas
- **JavaScript**: Funcionalidades dinâmicas do frontend

## 🚀 Funcionalidades

- Adicionar cadastro de novo usuário
- Listar e visualizar usuários cadastrados
- Editar dados do usuário existente
- Remover usuário do sistema
- Interface amigável e responsiva

## 📁 Organização do Projeto

- `/assets` — Arquivos de CSS, SCSS e JavaScript
- `/pages` — Páginas HTML/PHP (listar, editar, adicionar)
- `/includes` — Arquivos de conexão e funções auxiliares
- `/database.sql` — Script SQL do banco de dados
